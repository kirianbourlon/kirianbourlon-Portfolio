Contexte du projet :


La société BIO VISION récemment créée, opérant dans le secteur pharmaceutique est consciente qu’une infrastructure de réseau fiable est le fondement d'un puissant réseau et catalyseur pour la croissance et le succès dans toutes les entreprises.

Troisième Phase :

Une fois la topologie réseau mise en place, nous avons pu passer à la configuration de notre serveur AD (Active Directory), DHCP (Dynamic Host Configuration Protocol) et DNS (Domain Name System). Ce serveur est une machine virtuelle déployée via le logiciel VMWare. Toute la documentation relative à cette configuration est disponible dans ce PDF : PPE_Mission2.

- Première image : Vue de notre serveur AD avec les dossiers associés aux différents types d’utilisateurs, leurs comptes, leurs ordinateurs et leurs groupes.

- Deuxième image : Configuration des serveurs DNS, permettant de résoudre la zone directe BioVision.com ainsi que la zone inverse de l'adresse 172.20.0.10.

- Troisième image : Étendues DHCP configurées, montrant les plages d'adresses IP disponibles et les exclusions appliquées.

- Quatrième image : Présentation d’une machine sous Windows 11 classique ayant rejoint avec succès le domaine BioVision.com.

- Dernière image : Configuration réseau de la machine cliente, démontrant qu'elle fait partie du domaine BioVision.com et qu'elle utilise une adresse IP attribuée par le service DHCP.

CONCLUSION


Ce projet m'a permis de développer des compétences essentielles dans la conception et la mise en œuvre d'infrastructures réseau, tout en renforçant mes connaissances sur des technologies comme Cisco IOS, Active Directory, DHCP et DNS.
Il m'a offert une expérience concrète en résolution de problèmes et en gestion d'un projet technique complet, depuis l'analyse des besoins jusqu'à la configuration des équipements.
Cette expérience m'a également permis d'améliorer ma rigueur, mon sens de l'organisation et mes capacités à collaborer dans un contexte professionnel, des qualités indispensables dans le domaine des systèmes et réseaux.
